Block Till You Drop

A fast arcade survival game written in C++ using SDL2. Blocks fall continuously, and the player has to avoid being crushed while using powerups like bombs, lasers, and freeze tiles to clear space. This build includes cluster detection, falling physics, full-row clears, score tracking, and game over UI.

Features

Falling shapes with increasing speed

Cluster detection and falling block groups

Bomb, Freeze, Laser H + Laser V powerups

Movement, jumping, and ability cooldowns

Game over screen with high score tracking

Controls

A/D to move

Space to jump

Arrow keys to break tiles in that direction

R to restart
